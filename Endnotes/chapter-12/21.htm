Cato SASE Cloud; 2024;

Cato optimally secures all enterprise locations, users, applications, and clouds, into a global, cloud-native platform.

Cato can also be gradually deployed in order to replace various legacy security point solutions and legacy network services.

Catos platform features are:

1.      Optimized networking;

a.      Software Defined WAN (SD-WAN) (https://www.catonetworks.com/platform/software-defined-wan-sd-wan); and

b.      Global Private Backbone (https://www.catonetworks.com/cato-sase-cloud/global-private-backbone).

2.      Threat Prevention;

a.      Firewall-as-a-Service (FWaaS) (https://www.catonetworks.com/platform/firewall-as-a-service-fwaas);

b.      Secure Web Gateway (SWG) (https://www.catonetworks.com/platform/secure-web-gateway-swg);

c.      Intrusion Prevention System (IPS) (https://www.catonetworks.com/platform/intrusion-prevention-system-ips);

d.      Next-Generation Anti-Malware (NGAM) (https://www.catonetworks.com/platform/next-generation-anti-malware-ngam); and

e.      DNS Security (DNS) (https://www.catonetworks.com/platform/dns-security);

f.        Remote Browser Isolation (RBI) (https://www.catonetworks.com/platform/remote-browser-isolation-rbi).

3.      Data and App Protection;

a.      Cloud Access Security Broker (CASB) (https://www.catonetworks.com/platform/cloud-access-security-broker-casb);

b.      Data Loss Prevention (DLP) (https://www.catonetworks.com/platform/data-loss-prevention-dlp); and

c.      Zero Trust Network Access (ZTNA) (https://www.catonetworks.com/platform/universal-zero-trust-network-access-ztna).

4.      Detection and Response

a.      Extended Detection and Response (XDR) (https://www.catonetworks.com/platform/extended-detection-and-response-xdr); and

b.      Endpoint Protection (EPP/EDR) (https://www.catonetworks.com/platform/endpoint-protection-epp).

5.      Unified Management

a.      Management Application (CMA) (https://www.catonetworks.com/platform/cato-management-application-cma); and

b.      Platform API (https://www.catonetworks.com/platform/cato-api).

Refer to https://www.catonetworks.com/platform.
