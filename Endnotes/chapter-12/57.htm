Papermill; 2024;

Papermill is a framework and tool developed first by Netflix, and is now an open-source tool that allows you to run parameterized Jupyter notebooks via command line.  It is used mostly by those who want to run a notebook with different input parameters (like when conducting hyperparameter optimization.

Papermill is an example of a basic DataOps and MLOps tool. Since it focuses on operationalizing notebooks, which is an anti-pattern for high performing cost effective systems, the data engineer is cautioned to use it with caution and apply the best practices in this book.  It can be very useful for scientists to create workflows between notebooks.

Refer to https://github.com/nteract/papermill.
