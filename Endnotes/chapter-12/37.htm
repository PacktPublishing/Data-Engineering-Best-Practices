"D&B Direct+ Documentation"; by D&B; 2024;

D&B Direct is the API site for D&B.

In this document site these sections are presented to you:

1.       Getting Started is an introduction to D&B Direct+ with API like rate limiting, quota enforcement, versioning, best practices, and troubleshooting;

2.       Guides is a detailed area with information on: identity resolution; search; multi-process; enrichments like standard vs add-ons / non-standard data blocks); data file; research; monitoring; and FAQs;

2.      Resources contains resources on change history; data dictionary; error messages; ISO country codes supported; postman collections; JSON samples; monitoring sample deliveries; and multi-process samples;

3.      API Reference contains the detailed endpoints; requests; and responses for D&B APIs.

Refer to https://directplus.documentation.dnb.com.
