Secure Access Service Edge is a product group with the following components:

1.       Prisma SASE with ZTNA 2.0 security secures apps used by a hybrid workforce while securing access and data to reduce data breach risk;

2.      Prisma Access is a purpose-built secure cloud service that protects application traffic;

3.      Prisma Access Browser allows you to safely access SaaS and private apps from any device;

4.      Prisma SD-WAN ensures applications remain available based on real-time application performance SLAs and visibility.  It natively applies to branches with ZTNA 2.0, helping reduce breaches by 45%; and

5.      Autonomous Digital Experience Management (ADEM) is able to detect and resolve problems before impacts are felt . It uses AI to automate complex operations and accelerate incident detection / diagnosis.
