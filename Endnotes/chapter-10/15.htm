Veracode Dynamic Analysis; 2023; is used to scan web applications for vulnerabilities.

It is a unified web application scanning solution that dynamically scans, secures, and monitors web applications.

It combines scale and batch processing capabilities provided by Veracode DynamicMP (https://docs.veracode.com/r/c_using_dynamic_overview), with the ability to configure each application provided today by 'Veracode DynamicDS'.

Dynamic Analysis offers features that enable dynamic 'set it and forget it' capabilities for security teams via an authentication and configuration platform.

Refer to https://docs.veracode.com/r/Dynamic_Analysis.
