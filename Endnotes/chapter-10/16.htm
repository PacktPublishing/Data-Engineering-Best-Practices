"What Is DevSecOps? Adding Security to the SDLC"; November 29, 2021; by Nedim Maric;

'DevSecOps' is an approach that brings together development, security, operations, and 'infrastructure as code (IaC)' via CI/CD.

The 'DevSecOps' engineer is able to monitor, automate, and provide security compliance for SDLC stages.

When security testing and operations exists at each 'software development lifecycle (SDLC)' step, you can minimize security risks.

Refer to https://brightsec.com/blog/devsecops.
