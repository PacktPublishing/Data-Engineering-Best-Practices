Support Vector Machine; 2023;

Support Vector Machines or 'SVMs' are supervised learning models that analyze data for classification and regression analysis.

The algorithm maps training data to points in a vector space maximizing the width of the gap between categories.

New data is mapped into that space and predicted to belong to a category based on which side of the gap they fall (i.e., regression analysis).

SVMs can perform linear and non-linear classification using a ‘kernel trick.’ Find ‘Mathematics: the kernel trick’ on this page https://en.wikipedia.org/wiki/Kernel_method, by mapping inputs into high-dimensional feature spaces.

Refer to https://en.wikipedia.org/wiki/Support_vector_machine.
