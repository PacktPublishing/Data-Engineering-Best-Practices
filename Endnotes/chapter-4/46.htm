Gridgain; by Gridgain; 2023; GridGain combines:

1.    stream processing;

2.    a distributed in-memory data grid; and

3.    co-located compute that delivers data processing and low latency analytics at scale.

GridGain's distributed memory-first architecture and co-located compute enable millisecond latency data analytics process with disk-based persistence.

Gridgain is built on open-source Apache Ignite. With Gridgain computing, processing happens in RAM (not disk) for faster performance of an in-memory architecture.

For horizontally scalable clusters deployed on-premises and/or in public (or private clouds) provides a capability for handling demanding workloads.
