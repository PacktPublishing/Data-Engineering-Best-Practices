Cambridge Semantics; 2023; Cambridge Semantics is a company the produces the Anzo Platform (https://cambridgesemantics.com/anzo-platform). It is used to overcome the perceived limitations of other knowledge graphs with the capabilities and flexibility that complex, enterprise-scale solutions require.

The platform is a knowledge graph platform built on a high-performance graph database engine, called AnzoGraph (also on Amazon AWS) (https://aws.amazon.com/marketplace/pp/prodview-hdk62bf73ajma), that uses an in-memory MPP processing paradigm to execute queries against datasets extremely quickly, enabling agile data integration, transformation, and analytics at enterprise scale.

Anzo leverages standards including W3C's RDF, OWL, SKOS, and SPARQL to combine knowledge graphs of metadata and data which can be powerfully explored, transformed, and analyzed, while also ensuring open data interoperability and easy integration with other systems.

Anzo is an open overlay platform that allows users to assemble knowledge graphs against the underlying data resources without displacing or disrupting existing processes or platforms. Anzo integrates with enterprise metadata, governance, security controls and policies, and includes APIs for lights-out integration into other processes.

The Anzo Knowledge Graph (also on Microsoft Azure) (https://cambridgesemantics.com/anzo-platform) was on Azure but now has been removed. The concepts are central to the Data Fabric as an agile integration overlay that blends complex and diverse data into consumable data products, data scientists, and applications.

Anzo enables the implementation of the data fabric pattern by:

1.    automating redundant and labor-intensive tasks to build and manage the graph using metadata and AI;

2.    providing workflows supporting diverse consumers to manage the knowledge graph; and

3.    delivering enterprise scale with an MPP graph engine and cloud-based automation. See Anzo Data Fabric (https://info.cambridgesemantics.com/the-story-of-the-data-fabric-defined).
