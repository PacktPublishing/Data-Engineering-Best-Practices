StarRocks Documentation; 2023; StarRocks is a next-gen, high-performance analytical data warehouse that enables real-time, multi-dimensional, and highly concurrent data analysis.

StarRocks is built as 'massively parallel processing (MPP)' architecture and is:

1.    a fully vectorized execution engine;

2.    a columnar storage engine that supports real-time updates; and

3.    is powered by a set of capabilities like:

a)      a fully customized cost-based optimizer (CBO),

b)      intelligent materialized view and

c)      supports real-time and batch data ingestion from many data sources.

Directly analyzing data stored in data lakes with limited or no data migration is possible. StarRocks API is compatible with MySQL protocols to connect clients with third party business Intelligence (BI) tools.

StarRocks is built to be scalable, available, and maintainable. Use cases supported are:

1.    a variety of OLAP scenarios;

2.    real-time analytics;

3.    ad-hoc queries; and

4.    data lake analytics; etc;

Refer to: https://docs.starrocks.io/en-us/latest/introduction/StarRocks_intro.
