Azure Purview; 2023; Purview is a unified data governance solution to manage and govern: on-premises, multi-cloud, and software as a service (SaaS) data. You can:

1.    Create a holistic, up-to-date map of your data landscape with automated data discovery, sensitive data classification, and end-to-end data lineage; and

2.    Enable data consumers to access valuable, trustworthy data management.

Purview is a foundation for effective data governance and usage is available with Microsoft Purview Data Map. Key features are:

1.    Automate and manage metadata from hybrid sources;

2.    Classify data using built-in and custom classifiers and Microsoft Information Protection sensitivity labels;

3.    Label sensitive data consistently across SQL Server, Azure, Microsoft 365, and Power BI; and

4.    Integrate all your data catalogs and systems using Apache Atlas APIs.

Additional Features:

5.    Make data discoverable by using business and technical search terms;

6.    Dictionary and enterprise business glossary;

7.    Understand data origins with interactive data lineage visualization; and

8.    Provides data scientists, engineers, and analysts data needed for BI, analytics, AI, and machine learning.

Refer to: https://azure.microsoft.com/en-us/products/purview.
