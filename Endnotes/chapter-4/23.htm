HyperText Markup Language (HTML), versions prior to 2.0; 1990; HTML was originally developed at CERN by Tim Berners-Lee around 1990. In 1989, Berners-Lee wrote a proposal described as: 'an attempt to persuade CERN management that a global hypertext system was in CERN's interests'.

He was allowed to build the World Wide Web. HTML as a way of 'marking up' web documents.

CERN's HTML directions from 1992 stated, 'The HTML language has been in use in the field since 1990. '

From 1992 to 1995, Berners-Lee and Dan Connolly enhanced the HTML standard and in 1993 'Hypertext Markup Language (HTML)' was published as an Internet Engineering Task Force (IETF) Draft later enhanced in 1994.

HTML was based on SGML (ISO 8879:1986) with the addition of 'hypertext'.
