Victoria Metrics; 2023; Victoria Metrics is a fast, cost-effective scalable monitoring solution and time series database.

It is available as a binary release, docker image, snap packages with source code.

A cluster version is also available as well as a Victoria Metrics TSBS Benchmark (https://github.com/timescale/tsbs/blob/master/docs/victoriametrics.md).

Refer to: https://docs.victoriametrics.com.
