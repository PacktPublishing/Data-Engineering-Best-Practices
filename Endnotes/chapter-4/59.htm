MongoDB Time Series Collections; 2023; MongDB is the data engine time series data is a sequence of data points in which insights are gained by analyzing changes over time.

Time series data is generally composed of these components:

1.    Time when the data point was recorded;

2.    Metadata (ie source), which is a tag that uniquely identifies a series and rarely changes; and

3.    Measurements (metrics or values), which are the data points tracked at increments in time.

Time series data is stored as key-value pairs which change over time. Time series collections store this time series data. In MongoDB time series collections … writes operations are organized so data from the same sources and times are stored alongside each.

A MongoDB TSBS Benchmark exists for it (https://github.com/timescale/tsbs/blob/master/docs/mongo.md).

Refer to: https://www.mongodb.com/docs/manual/core/timeseries-collections.
