AWS Glue; 2023;  is a serverless data integration service for analytics users to discover, prepare, move, and integrate data from multiple sources.

It is used for analytics, machine learning, and application development. It includes productivity and data operations: DataOps (https://www.techtarget.com/searchdatamanagement/definition/DataOps) tooling for authoring, running jobs, and implementing business workflows.

One can discover and connect > 70 data sources and manage data sets in a centralized data catalog. You can:

1.    visually create,

2.    run, and

3.    monitor

extract, transform, and load (ETL) pipelines to load data into your data lakes.

One can immediately search cataloged data using Amazon Athena (https://aws.amazon.com/athena), Amazon EMR (https://aws.amazon.com/emr), and Amazon Redshift Spectrum (https://docs.aws.amazon.com/redshift/latest/dg/c-getting-started-using-spectrum.html).

Refer to: https://docs.aws.amazon.com/glue/latest/dg/what-is-glue.html.
