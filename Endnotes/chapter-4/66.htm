Athena is a serverless, interactive analytics service built on Trino (https://trino.io), PrestoDB (https://prestodb.io), and Apache Spark (https://spark.apache.org) frameworks.

Athena can be used to run SQL queries at petabyte scale on Amazon Simple Storage Service (S3) (https://aws.amazon.com/s3).
