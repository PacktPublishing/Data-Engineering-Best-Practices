Microsoft Graph; 2023; Microsoft Graph is the gateway to data and intelligence in Microsoft 365 (now Office 365) (https://www.office.com), and it provides a unified programmability model to access the tremendous amounts of data in Microsoft 365, Windows, and Enterprise Mobility + Security.

Data in Microsoft Graph can be used to build applications for consumers that interact with millions of users.

At the core of Microsoft Graph are the concepts of the user and group so it has a primary focus with graph oriented security service integration.

Refer to: https://learn.microsoft.com/en-us/graph/overview.
