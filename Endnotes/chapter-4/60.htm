QuestDB; 2023; QuestDB is a highly performant open-source time series database supporting SQL with time-oriented extensions.

It implements:

1.    The PostgreSQL wire protocol;

2.    A REST API; and

3.    QuestDB.

It supports ingest operations using the InfluxDB (https://www.influxdata.com) line protocol.

A QuestDB TSBS Benchmark exists for QuestDB (https://github.com/timescale/tsbs/blob/master/docs/questdb.md).

Refer to: https://questdb.io.
