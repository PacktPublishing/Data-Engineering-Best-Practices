Google Pub/Sub Dataflow Examples; 2024;

1.      Write data from Kafka to BigQuery with Dataflow; 2023; is an example of how to  create and deploy a Dataflow Error! Bookmark not defined. pipeline to stream from Apache Kafka into Google BigQuery. See https://cloud.google.com/dataflow/docs/kafka-dataflow.

2.      Streaming with Pub/Sub; 2023; is an overview of Dataflow Error! Bookmark not defined. integration with Google Pub/Sub. You will want to learn about Apache Beam concepts first.  See  https://cloud.google.com/dataflow/docs/concepts/streaming-with-cloud-pubsub.

3.      Use Pub/Sub Seek with Dataflow; 2023; Pub/Sub Seek lets a consumer replay and then reprocess previously acknowledged messages or to acknowledge messages in bulk. Dequeuing messages is different between cloud provider and third party tooling.  Refer to https://cloud.google.com/dataflow/docs/guides/using-cloud-pubsub-seek.  In this example … the advice given by Google is:

a.      that Google does not recommend direct access to Pub/Sub Seek from within a running Dataflow pipeline. Direct access invalidates Dataflows watermark logic and does not work well with exactly once processing.

b.      in addition, direct access conflicts with the state of a pipeline that incorporates processed data.

c.      that Google recommends Pub/Sub Seek be used with this workflow:

                                                                          i.      make a snapshot of the subscription;

                                                                         ii.      drain the subscription messages in Dataflow; and

                                                                       iii.      resubmit the pipeline.
