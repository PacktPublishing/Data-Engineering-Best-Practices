InfoSum; 2024;

InfoSum produces a Data Collaboration Platform to run analytics across internal and external datasets. Data collaboration enables your organization to protect data security and customer privacy.

Cloud Vaults eliminate data copy often needed for collaboration, and safely use first-party data that exists across a network of partners.

Data Transformation allows for multi-dimensional datasets to be integrated.

Bunkers are DCRs that remove personal data (PII) for collaboration.

Safe Audience Transfer allows unlimited Bunkers to be connected without data movement.

Enhanced Differential Privacy Techniques control size / granularity of all results to guarantee that insights never identify a single individual or set of individuals while still providing valuable insights.
