StreamSets; 2023;

StreamSets is a Software AG company that aims to make data integration frictionless in complex hybrid and multi-cloud environments.

Data is unlocked - without ceding control thus enabling the modern data-driven enterprise.

Data pipelines deliver real-time analytics data for decision support.

StreamSet is a framework that allows you to spend more time building data smart pipelines, enabling self-service innovation.

StreamSets Data Collector Engine is a data pipeline engine for streaming, change data capture (CDC) and batch ingestion from various sources to various destinations.

The platform supports some common use cases – see the StreamSets Data Sheet (https://streamsets.b-cdn.net/wp-content/uploads/StreamSets-Data-Sheet.pdf).

1.      Agile Reporting: leverages enterprise data to improve consistency, accuracy, and depth of reporting.  Various file sources are leveraged as well as a wealth of legacy data source connectors.

2.      Advanced Analytics: integrates enterprise legacy analytic data stores with cloud data platform capabilities to enable analytics and decision support.

3.      Operational Analytics: processes data in route from the OLTP source to cloud data store, so analytics ready data is available for operations.
