Amazon EC2 High Memory Instances; by AWS; 2025;

Refer to:

1.      AWS Instance Type U1  instances are built on AWS Nitro System with up to 24TiB of memory in a single instance. They come in 3, 6, 9, 12, 18, and 24 TB sizes with up to 38 Gbps of dedicated storage bandwidth.

2.      AWS Instance Types U7i instances offer up to 45% better price performance over existing Amazon EC2 High Memory (U-1) instances. U7i instances come in 6, 8, 12, 16, 24, 32TiB (U7inh) sizes. The U7i instance type is powered by 1.9 GHz (Turbo Boost to 2.90 GHz) 4th Generation Intel Xeon Scalable processors (Sapphire Rapids) and support up to 1920 vCPUs.

Refer to https://aws.amazon.com/ec2/instance-types/high-memory.
