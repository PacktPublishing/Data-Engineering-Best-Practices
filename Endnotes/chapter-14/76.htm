AWS DataSync; by AWS; 2024;

AWS DataSync is a service to automate the movement of data between on premises and AWS Storage services.

Supported copy targets and destinations are:

1.      Network File System (NFS) shares;

2.      Server Message Block (SMB) shares;

3.    Hadoop Distributed File Systems (HDFS);

4.      Self-managed object storage;

5.    AWS Snowcone;

6.      Amazon S3 buckets;

7.      Amazon Elastic File System (Amazon EFS) file systems;

8.      Amazon FSx for Windows File Server file systems;

9.      Amazon FSx for Lustre file systems;

10.  Amazon FSz for OpenZFS file systems; and

11.  Amazon FSx for NetApp ONTAP file systems.

Refer to https://aws.amazon.com/datasync.
