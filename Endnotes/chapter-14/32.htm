Snowflake Distributed DCR; 2024;

Snowflake Distributed DCR is a concept marketed on the following products:

1.    Snowflake Data Sharing;

2.    Snowflake Private Data Exchange platform (i.e., Snowflake for Collaboration (https://www.snowflake.com/en/data-cloud/workloads/collaboration);

3.    Snowflake Secure Function (UDF) (i.e., Protecting Sensitive Information with Secure UDFs and Stored Procedures (https://docs.snowflake.com/developer-guide/secure-udf-procedure) and;

4.    Secure join, underlying multi-tenant cloud data platform and other Snowflake services needed to provide a foundation to run a data clean room.

As you can see a lot needs to go into a DCR and no mention of the core data encryption problem that Decentriq raises to add the needed value.

Refer to https://www.snowflake.com/blog/distributed-data-clean-rooms-powered-by-snowflake.
