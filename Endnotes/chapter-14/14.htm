Azure Stream Analytics Examples; 2023;

Azure Stream Analytics examples include:

1.      Tutorial: Capture Event Hubs data in parquet format and analyze with Azure Synapse Analytics (https://learn.microsoft.com/en-us/azure/stream-analytics/event-hubs-parquet-capture-tutorial);

2.      Tutorial: Write to a Delta Table stored in Azure Data Lake Storage Gen2 (https://learn.microsoft.com/en-us/azure/stream-analytics/write-to-delta-table-adls-gen2);

3.      Github: Generate Fluent Lite from Streamanalytics (via Azure SDK for Java) (https://github.com/Azure/azure-sdk-for-java/tree/main/sdk/streamanalytics/azure-resourcemanager-streamanalytics/src/samples/java/com/azure/resourcemanager/streamanalytics/generated); and

4.      K2 Academy: Azure Stream Analytics (https://k21academy.com/microsoft-azure/data-engineer/azure-stream-analytics). See Figure 14.3 - 2023 K2Academy - How to Send Data to Azure Stream Analytics below where the use cases of Azure Stream Analytics is defined.

a.      Store streaming data to make it available for other cloud service analysis &  reporting;

b.      Transform and analyze in real-time;

c.      Provide real-time dashboarding via Power BI (aka for monitoring);

d.      Send alerts to subscribers;

e.      Enable decision processing in real-time;

f.        Application of machine learning models for risk analysis, fraud detection, and predictive trend generation (note: advanced analytics and ML model generation support is limited); and

g.       Provide for geospatial analytics for driverless vehicle fleet management.
