Decentriq; 2023;

The Decentriq approach to the creation of data clean rooms adds a lot of value to AWS and GCP cloud provider offerings for Data lake / warehouse providers like Snowflake and Databricks.

It enables you to build out a confidential compute capability in data clean rooms (DCR) with a focus on encryption of data that is not similarly present in the raw cloud providers services.

Notice that Microsoft's Azure is omitted from the Decentriq offering. Snowflake and Databricks relied on their framework of standard computations and cloud vendors strict access controls in order to secure data. Without data encryption, data remains mutually trusted between cloud provider, other consumers, and the customer.

And customers are exposed to some very significant privacy / security risks.

Guaranteeing privacy through data encryption is necessary to secure sensitive information.
