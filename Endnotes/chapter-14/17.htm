TIBCO Spotfire Cloud Data Streams; 2024;

TIBCO Cloud Data Streams software brings the power of real-time analytics to the cloud as a zero-install, browser-based, cloud-based, wizard-driven, real-time streaming business intelligence (BI) platform.

Spotfire Streams provides for real-time assessment, detection, and action with streaming analytics.  It provides for integration and quick adoption for the developer and the enterprise with dynamic learning. Spotfire® Streaming is immediate.  It applies learning algorithms to live streaming data in real time and is designed for insight generation.  It can be embedded in applications to automate decisions, or It can be placed into operational dashboards.  A offers continuous query processing engine supports live real-time data; so, it may be combined with historical data through native or custom data connectors. Note:

"Spotfire Data Streams is an assembly of TIBCO software technologies in a single licensable collection, configured as an add-on for Spotfire licensees. Spotfire Data Streams is a subset of the TIBCO Streaming package, configured for the sole purpose of serving live data to TIBCO Spotfire." - Source: What is TIBCO Spotfire Data Streams?
