TIBCO Spotfire Data Streams; 2024;

Spotfire Streams is a streaming data facility for Tibco Spotfire Analytics to analyze and visualize real-time data.

A continuous query processing engine supports live real-time data; so, it may be combined with historical data through native or custom data connectors.

Note:

"Spotfire Data Streams is an assembly of TIBCO software technologies in a single licensable collection, configured as an add-on for Spotfire licensees. Spotfire Data Streams is a subset of the TIBCO Streaming package, configured for the sole purpose of serving live data to TIBCO Spotfire." - Source: What is TIBCO Spotfire Data Streams? (https://docs.streambase.com/sb16/index.jsp?topic=%2Fcom.streambase.sb.ide.help%2Fdata%2Fhtml%2Fwelcome%2Fstream-comparison.html)
