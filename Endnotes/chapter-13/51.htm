Cambridge Semantics Anzo Platform (now Altair Graph Studio); 2023; Anzo is a scalable knowledge graph platform for data integration and analytics. It has a unique capability to support an ontological model in OWL-RDF and Labeled Property Graph (LPG) in a single platform over the same physical knowledge base objects.

Note that the "Cambridge Semantics Anzo Platform" is now known as "Altair Graph Studio" following the acquisition of "Cambridge Semantics" by "Altair Engineering" (effective on 2024-12-10). The Anzo platform is now rebranded into the Altair product suite. The same core functionality with the knowledge graph platform is available as is semantic data integration with the prior analysis capabilities.
