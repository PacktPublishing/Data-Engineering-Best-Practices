Oracle® Retail Insights; 2023; Release 17.0 - E95062-02; Oracle Retail Insights is a reporting facility using the Retail Insights repository in the Oracle Business Intelligence (Oracle BI) suite. Dimensions, attributes, metrics, and terminology of the metadata is available. Several data sources are leveraged including Oracle Retail Merchandising System (RMS) and Oracle Retail Price Management System (RPM). Data is extracted, loaded, and transformed into the Retail Insights data model for reporting. Retail Insights is augmented with your data using the packaged Oracle Data Integrator ETL programs.

“Retail Insights dimensions and attributes represent the structure and activities of a retail organization and make measurement possible.” - Oracle

Refer to: https://docs.oracle.com/cd/E69213_01/pdf/170/html/user_guide/dimensions_attributes.htm.
