How to use a semantic layer for data and analytics - A Practical Guide; 2012; AtScale. The use of cloud data platforms does not address the fundamental need to get actionable data to the consumers. Building a semantic layer business case has three focus areas:

1.      Increase data value by facilitating data set combination of large data sets, with a wider breadth of coverage, to make the total set more accessible to consumers.

2.      Reduce the delivery cost of prepared, cleansed, and catalogued data in a cloud optimized manner that simplifies data engineering (DataOps) to promote consumer productivity.

3.      Increase number and types of data consumers since it easier would be easier to interact with semantic enriched data. Improving the knowledge about data enables more data driven insights to be generated.
