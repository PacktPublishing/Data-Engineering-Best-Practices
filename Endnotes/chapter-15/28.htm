R Notebooks; 2024;

in R Studio you can create a new notebook via

Rstudio -> File -> New File -> R Notebook

This opens a default notebook with prefilled text.

To create a notebook, begin with:

~/.config/rstudio/templates

and save a copy of the default notebook.Rmd with a change to the title.

This shows its origin, rather than the default path:

 / Applications / RStudio.app / Contents / Resources / app / resources / templates

The default path was created by the installation of RStudio.app

Creating a new notebook uses a template from

~/.config/rstudio/templates.

Note the opening a notebook will show the template it was created from. Any template applied by the new notebook file option has to be named notebook.rmd; since notebook_osx.Rmd does not work.

Refer to https://community.rstudio.com/t/new-r-notebook-default-template-notebook-osx-rmd/157576.
