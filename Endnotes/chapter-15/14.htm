Count.co; 2024;

Count.co helps data teams solve complex problems.

It is a collaborative canvas that allows you to explore data, tell stories and build models via a real-time whiteboard.

The count canvas is different from traditional BI tools since it unifies the analytics process and provides a place for requirements gathering, prototyping, query writing, data modeling, visualization, reporting, and documentation.

The count canvas is a real-time, multiplayer environment where  all see each other's changes as they are being made.

Importing from Jupyter notebooks is supported.

Refer to https://count.co.
