IBM Databand; by IBM; 2024;

IBM Databand is a data observability product with the following:

1.     Automatically collects metadata so you have a standard set of methods for data quality validation.

2.     Creates historical baselines based on common data behaviors, essentially profiling your data pipeline.

3.     Alerts on anomalies as determined by rules that pinpoint deviations relative to the data's profile or rule detected violations.

4.     Remediates notification workflows to resolve data quality issues and keep data set deliveries on track.

Refer to https://www.ibm.com/products/databand.
