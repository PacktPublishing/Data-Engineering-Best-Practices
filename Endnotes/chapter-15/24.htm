Noteable; 2024;

Noteable was a cloud-based notebook used for data science, ML and analytics. A number of comprehensive features existed including the fact that is it library agnostic so any Python code, including TensorFlow and PyTorch can be leveraged. It integrates with Dagster, Airflow and Papermill for orchestration.

Noteable has been shutdown (2023-12-31) It has many of the same capabilities as the Code Interpreter plugin. It was acquired by Confluent (https://www.confluent.io/blog/welcoming-noteable-to-confluent/). In the announcement you can find the following quote:

"We are thrilled to join Confluent and propel data in motion. We both share a commitment to open standards, supporting technical communities, and contributing to open source. We’ll focus on simplifying data streaming to make it accessible to all data practitioners and therefore continuing with our vision to democratize data within organizations."

Source Pierre Brunelle, CEO & Co-founder, Notable

Note further that the OpenAI Code interpreter also has a custom model that could someday plug into notebooks.
