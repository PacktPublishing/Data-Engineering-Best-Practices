RMarkdown; 2024;

RMarkdown files are designed to be used with the RMarkdown package.

The package comes installed with the RStudio IDE, but can be obtained from CRAN with the command:

install.packages("rmarkdown");

See also: RMarkdown Rreference Guide (https://www.rstudio.com/wp-content/uploads/2015/03/rmarkdown-reference.pdf).

Refer to https://rmarkdown.rstudio.com/lesson-1.html.
