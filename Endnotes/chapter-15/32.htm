Starboard; 2024;

Starboard is a browser based notebooks similar to Jupyter notebook.

It provides a quick way to visualize data interactively. It is a rapid prototyping tool and used to build dashboards.

Cell-by-cell notebooks are made available in the browser.

WebAssembly (https://webassembly.org) makes it possible to run a number of languages in the browser with acceptable performance (e.g., Python through Project Pyodide - https://github.com/pyodide/pyodide).

Refer to https://starboard.gg.
