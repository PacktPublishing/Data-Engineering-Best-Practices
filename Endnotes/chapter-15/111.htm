Set Your Jupyter Notebook up Right with this Extension; by Will Koehrsen (https://williamkoehrsen.medium.com in Towards Data Science); March 1, 2019;

The approach laid out address notebook problems like:

1.      No title;

2.      No explanations of what the code should do or how it works;

3.      Cells run out of order; and

4.      Errors in cell output.

The solution is to use a Jupyter notebook extension that on opening a new notebook automatically:

5.      Creates a template that guides a notebook author to produce structured documentation;

6.      Inserts a number of common library imports / settings; and

7.      Prompts the author to change the notebook name from the default label which is Untitled.

The extension changes the notebook defaults settings.

This way markdown cells, notebook name, and imports are mandatory features of the new notebook rather than the out-of-the box default (which has none of these set).
