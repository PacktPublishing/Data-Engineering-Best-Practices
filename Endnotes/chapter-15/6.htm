Anaconda; 2024;

Anaconda is an open source distribution of the Python and R programming languages for scientific computing, that simplifies package management and deployment. Packaged versions are managed by the package management system call conda. Anaconda Team Notebooks (https://www.anaconda.com/team-notebooks) is JupyterHub combined with authentication, security, and management capabilities of the Anaconda Server.

It provides you with a method of managing a JupyterHub instance using the tools that power the Anaconda Server.

Within Anaconda Team Notebooks you can:

1.      Create / manage groups of users;

2.      Assign user roles; and

3.      Supply users with custom channels populated with safe, Anaconda curated packages, based on specific need.

Refer to https://www.anaconda.com.
