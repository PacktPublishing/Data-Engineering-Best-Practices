Jupyter Interpreter for Apache Zeppelin; 2024;

This Zeppelin interpreter is a bridge between Zeppelin and the Jupyter kernel engine.

You can use Jupyter kernels if you also install the correct dependencies.

The steps are as follows:

1.      Install the Jupyter kernel desired from the available list of kernels located here:

https://github.com/jupyter/jupyter/wiki/Jupyter-kernels

2.      Find the kernel name by running  this command:

bash jupyter kernelspec list

3.      Run the kernel by running  this command:

%jupyter(kernel=kernel_name)
