OpenLineage; 2024;

Data lineage enables consistent collection of lineage metadata, creating a deeper understanding of how data is produced and used.

OpenLineage is an open platform for the collection and analysis of data lineage.

Metadata is tracked for various datasets, jobs, and job runs giving users information to identify a data issues root cause. Understanding the impact of change is also important.

OpenLineage is an open standard for data lineage collection with a metadata repository reference implementation (Marquez), as well as libraries for common languages, and integrations with data pipeline tools.

Refer to https://openlineage.io.
