Keboola - Data Lineage; 2024;

Keboola provides complete data flow visualization and capability to run a data service without writing code.

The platform automates the data pipeline from a drag-n-drop GUI, by collecting data, transforming it, and storing it for analysis.

Keboola's main capability is automating data pipelines with tracing data lineage a key delivery of that automation.

Data operations are tagged with metadata describing the following:

1.      user activity;

2.      job activity;

3.      data flow;

4.      schema evolution;

5.      data pipeline performance; and

6.      compliance with security rules … etc

Keboola is integrated with OpenLineage.

Refer to https://www.keboola.com/product/data-lineage.
