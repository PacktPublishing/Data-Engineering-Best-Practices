JetBrain's Rider; 2024;

JetBrain's Rider is a fast and powerful cross-platform .NET IDE .

It supports the Microsoft .NET Framework, the new cross-platform Microsoft .NET Core, and Mono (an open-source . NET Framework-compatible software framework) based projects.

Applications like .NET desktop, services and libraries, Unity games, Xamarin apps, ASP.NET, and ASP.NET Core web applications can be created.

There are >2200+ live code inspections, hundreds of context actions and refactoring's available in by ReSharper.

JetBrain's Rider supports cross-platform running: Windows, macOS, and Linux.
