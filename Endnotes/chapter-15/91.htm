Jupyter Kernels; 2024;

In this link you can see a listing of Jupyter notebook kernels available currently.

Note that kernel zero is the IPython Kernel, which you can get through ipykernel (https://pypi.org/project/ipykernel).

The ipykernel is a reference implementation for other kernels.

Refer to https://github.com/jupyter/jupyter/wiki/Jupyter-kernels.
