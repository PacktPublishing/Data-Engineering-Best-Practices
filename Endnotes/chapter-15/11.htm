DataRobot Notebooks; 2024;

DataRobot Notebooks are part of the DataRobot AI Platform (https://docs.datarobot.com/en/platform) and they are available in both DataRobot Workbench and DataRobot Classic.

They offer flexibility to develop and run code in the language of your choice using both preferred open-source ML libraries as well as the DataRobot API for streamlining and automating your DataRobot workflows.

Also, you can organize, collaborate, and share notebooks assets in a unified environment with centralized governance via DataRobot Use Cases.

DataRobot Notebooks are Jupyter compatible, and they use Jupyter kernels, and have import and export capability to the .ipynb file format.

Refer to https://docs.datarobot.com/en/docs/dr-notebooks/index.html.
