Microsoft Azure Notebooks; 2024;

With Microsoft Azure Notebooks you can enjoy powerful, integrated Jupyter notebooks with the following products and services from Microsoft and GitHub:

1.    Notebooks in Visual Studio Code - To make the transition easier from Azure Notebooks, Microsoft has made the container image available so it can use with VS Code;

2.      GitHub Codespaces - Provides cloud-hosted environments where you can edit notebooks using VC Code or your web browser and store them on github.  Codespaces offers a Jupyter notebook experience as VS Code, without needing to install anything. It also allows you to use a cloud compute of choice;

3.      Azure Machine Learning - Provides a machine learning platform to enable users to build / deploy models on Azure. Azure ML allows you to run notebooks on a virtual machine or a shared cluster computing environment;

4.      Azure Lab Services - allows educators to provide on-demand access to preconfigured virtual machine s for a classroom;

5.      GitHub - Provides a source control way to store, collaborate and share notebooks with others.

To learn about how to: Run Jupyter notebooks in your workspace read here: https://learn.microsoft.com/en-us/azure/machine-learning/how-to-run-jupyter-notebooks?view=azureml-api-2; also, refer to https://visualstudio.microsoft.com/vs/features/notebooks-at-microsoft.
