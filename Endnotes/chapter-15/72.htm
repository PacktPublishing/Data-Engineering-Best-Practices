"How does Holistics modeling layer work with non-additive measures?"; by Holistics; 2024;

Holistics is a business intelligence platform to enable self-service data access for enterprises.

Holistics translates business operations into SQL queries and runs them on the SQL enabled database.

This is done via a modeling layer where metrics and data logic are defined.

Data team use analytics-as-code languages (AMQL) to define the data model.

 

"Holistics modeling can work well with non-additive measures when you have the raw data. Since Holistics can perform just-in-time SQL generation based on the list of fields the user select, instead of requiring pre-aggregation of data."

 

Refer to https://docs.holistics.io/docs/modeling/non-additive-measures.
