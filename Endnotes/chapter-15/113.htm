NBStripout; 2024;

nbstripout reads a notebook, strips output (you will want to make sure it does not strip essential metadata), and then writes a cleansed version back to the original file.

This tool is to be used as a git filter or pre-commit hook for users who don't want to track its output.

It's like the clear all output notebook command. It effectively allows you to keep output in the file on disk, but it does not commit the output to git.

Refer to https://github.com/kynan/nbstripout.
