Code Prettify; 2024;

Code Prettify is a Jupyter notebook extension that prettifies notebook code cells. It uses a call to the current notebook kernel to reformat code.

The prettifier package has to be callable from the current kernel language.

The nbextension provides:

1.      A toolbar button;

2.      A keyboard shortcut for reformatting a current code-cell;

3.      A keyboard shortcut for reformatting the entire notebook.

The extension points out incorrect syntax for correction.

Refer to https://jupyter-contrib-nbextensions.readthedocs.io/en/latest/nbextensions/code_prettify/README_code_prettify.html.
