"Use Ray on Databricks"; 2024;

Ray is an open-source framework used to scale AI and Python applications.

There is a compute layer supporting parallel processing.

Your applications will need to be compliant with the Ray Core (https://docs.ray.io/en/latest/ray-core/walkthrough.html) that defines core primitives (i.e., tasks, actors, objects) for building / scaling distributed applications. See: Ray (https://docs.ray.io/en/latest).

Refer to https://docs.databricks.com/en/machine-learning/ray-integration.html.
