Matplotlib;2024;

Matplotlib is a library for creating: static, animated, and interactive visualizations in Python. You can do the following:

1.      Create quality plots;

2.      Create interactive figures (with zoom, pan, update);

3.      Customize visual styles (https://matplotlib.org/stable/gallery/style_sheets/style_sheets_reference.html) and layouts (e.g., Mosaics: https://matplotlib.org/stable/users/explain/axes/mosaic.html);

4.      Export to various file formats (see 'matplotlib.figure.Figure.savefig' on https://matplotlib.org/stable/api/figure_api.html;

5.      Embed in JupyterLab and GUIs; and

6.      Use a many third-party packages built on Matplotlib (https://matplotlib.org/mpl-third-party).

Refer to https://matplotlib.org.
