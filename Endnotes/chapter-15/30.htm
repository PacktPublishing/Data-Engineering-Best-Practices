SAS Notebooks (How to run SAS programs in Jupyter Notebook); April 4, 2016;

Within Jupyter, the sas_kernel provides multiple ways to access SAS programming methods.

Two ways to run SAS code are:

1.      Create a new SAS notebook (from the New menu in the Jupyter home window and the File menu in an active SAS notebook, you enter and run SAS code directly from a cell.

2.      Use special Jupyter magic supported by the sas_kernel See https://github.com/sassoftware/sas_kernel. These magic commands look like SAS macro calls from a Python notebook. You inject SAS program code and pull in SAS results from the notebook.

Refer to https://blogs.sas.com/content/sasdummy/2016/04/24/how-to-run-sas-programs-in-jupyter-notebook.
