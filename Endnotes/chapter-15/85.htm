Dremio - Get Started; 2024;

Dremio provides a query engine capability to access your data in Data Lakes, such as on Amazon Web Services (AWS) or Microsoft Azure.

Also offered is a seamless lakehouse data management capability that is built on Icebergs data lakehouse for management and GIT like data versioning of datasets.

The virtualized integration to your data sources is managed across clouds with various integrations.

A key capability is Dremio's support for self-service analytics.

Refer to https://www.dremio.com/get-started.
