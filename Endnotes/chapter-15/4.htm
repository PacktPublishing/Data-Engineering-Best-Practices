"Operationalize your Amazon SageMaker Studio notebooks as scheduled notebook jobs"; by AWS; 2024;

Amazon SageMaker Studio has a managed solution for data scientists to enable data workers to run notebooks as ephemeral jobs.

This removes the need to recode Python modules or learn DevOps software development best practices to operationalize their solutions.

The problem with this approach is that once the notebooks enter the production stream who maintains them overtime.

The Data Engineer but note that role has been cut out of the deployment, DevOps and DataOps software lifecycle and that will cause business interruptions in the future.

Refer to https://aws.amazon.com/blogs/machine-learning/operationalize-your-amazon-sagemaker-studio-notebooks-as-scheduled-notebook-jobs
