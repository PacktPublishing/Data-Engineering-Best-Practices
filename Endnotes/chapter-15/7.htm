Apache Zeppelin; 2024;

Apache Zeppelin is a collaborative browser-based notebook for data analytics.

Apache Zeppelin dynamically creates input forms depending the backend language.

An interpreter is a plug-in that enables use of a specific language/data-processing-backend (e.g., to use Scala code you use the %spark interpreter which also covers Spark, PySpark, IPySpark, SparkR and Spark SQL).

There are three types of interpreters as follows:

1.      Community Managed Interpreters (https://zeppelin.apache.org/docs/0.11.0/usage/interpreter/installation.html#community-managed-interpreters);

2.      3rd Party Interpreters (https://zeppelin.apache.org/docs/0.11.0/usage/interpreter/installation.html#3rd-party-interpreters); and

3.      Available Community Managed Interpreters (https://zeppelin.apache.org/docs/0.11.0/usage/interpreter/installation.html#available-community-managed-interpreters).

Refer to https://zeppelin.apache.org.
