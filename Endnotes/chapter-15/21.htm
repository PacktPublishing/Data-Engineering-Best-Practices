Kaggle (Kernel) Notebook; 2024;

Kaggle Notebooks is a Jupyter notebook server that integrates enhanced processing.

In the notebook setting's tab, you can toggle: language; Docker image selection; Internet (the default); and an accelerator: CPU (default), GPU, or TPU.

The notebook allows for cloud hosted ML operations (rather than using local resources).

Refer to https://www.kaggle.com/docs/notebooks.
