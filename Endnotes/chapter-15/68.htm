Variable Inspector; 2024;

Variable Inspector is a Jupyter extension, which supports Python and R notebook kernels, to collect all variables, types, memory size,  and content in a window.

Displayed variables are automatically updated during cell execution.

Variables may be removed by clicking a link.

Position and display state (e.g., displayed/collapsed) are stored in metadata and later restored on restart.

Refer to https://jupyter-contrib-nbextensions.readthedocs.io/en/latest/nbextensions/varInspector/README.html.
