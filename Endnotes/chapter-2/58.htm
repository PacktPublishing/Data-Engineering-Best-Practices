Xplenty; 2024; has been rebranded as Integrate.io (https://www.integrate.io). Integrat.io was launched on December 15th, 2021. It combined four companies (Integrate.io, DreamFactory, FlyData and Intermix.io) into a single data integration platform. The new platform overview is as follows:

DreamFactory has API generation capabilities enabling API access to various consumable data sources in enterprise data systems.

FlyData is a CDC and ELT platform that replicates enterprise data into a data warehouse enabling access to real-time data normally locked up behind data silos.

Intermix.io has data warehouse analytic capabilities to expose data warehouse data and metadata for analytics use cases.

Xplenty is a low-code ETL / reverse ETL platform to streamline data preparation processing leading to insights Xplenty is a cloud based ETL solution that enables visualized data pipelines for automated data flows across several data sources and destinations.  It contains platform-based transformation tools that allow you to clean, normalize, and transform data while adhering to compliance best practices.  The important Xplenty features are:

1.    a powerful, no-code, on-platform data transformation rest API connector

2.    ability to pull in data from various sources that have a rest API destination

3.    the flexibility to send data to databases, data warehouses, and Salesforce

4.    a security focus for field-level data encryption and masking to meet compliance needs

5.    a rest API to achieve any other capability of the platform

6.    an Xplenty UI via the Xplenty API

7.    a customer centric human support team.

Refer to: https://www.integrate.io/product/etl.
