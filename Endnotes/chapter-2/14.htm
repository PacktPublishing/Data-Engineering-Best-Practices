AWS Database Migration Service (DMS); 2023;  is a managed service that moves databases and analytics workloads to AWS quickly, and securely with minimal downtime. AWS DMS supports migration between >20 database and analytics engines.  Examples:

1.    Oracle to Amazon Aurora MySQL-Compatible Edition,   

2.    MySQL to Amazon Relational Database (RDS) for MySQL,   

3.    Microsoft SQL Server to Amazon Aurora PostgreSQL-Compatible Edition,   

4.    MongoDB to Amazon DocumentDB (with MongoDB compatibility), 

5.    Oracle to Amazon Redshift, and  

6.    Amazon Simple Storage Service (S3).

Refer to: https://aws.amazon.com/dms.
