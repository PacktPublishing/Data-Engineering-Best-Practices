Alooma (now owned by Google); 2023; provides data pipeline as a service. The tool is used to ETL data into BigQuery, Redshift, and Snowflake for AI, ML, analytics, BI, and analytical reporting use cases. Offered is a code engine, data mapper, a restream queue for error processing, and a live data visualization tool for data stream monitoring. Note that the Alooma product will continue to support other cloud providers after its Google acquisition. It will only be accepting new customers that are migrating data to Google Cloud Platform (GCP), but existing customers will continue to have access to other cloud providers. Refer to: https://techcrunch.com/2019/02/19/google-acquires-cloud-migration-platform-alooma.
