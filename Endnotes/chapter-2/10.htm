AirFlow; 2023; is an open-source platform for developing, scheduling, and monitoring batch-oriented workflows. It has an extensible Python framework to build workflows connecting with various technology. A web interface manages workflow states. It is deployable from a single process on your laptop or via distributed setup supporting big workflows. Workflows are defined in Python code as Workflows as code and they serve a number of purposes:

1.    dynamic pipelines allowing for pipeline generation;

2.    extensible framework operators to connect with various technologies to adjust to your environment; and

3.    flexible parameterization via the Jinja Templating Engine.

Refer to: https://airflow.apache.org/docs/apache-airflow/stable/index.html.
