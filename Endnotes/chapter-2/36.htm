Oracle Data Integrator; 2023;  is a data integration platform with the ability to handle high-volume, high-performance batch loads as well as event-driven, trickle-feed integration processes and Service Oriented Architecture (SOA) enabled data services. ODI12c (latest version) provides improved user experience and redesigned flow-based declarative GUI and deep integration with Oracle GoldenGate. ODI12c adds parallelism when executing data integration processes, and interoperability with Oracle Warehouse Builder (OWB).  Operationally, Additionally, ODI12c  may be monitored from Oracle Enterprise Manager 12c. Refer to: https://www.oracle.com/middleware/technologies/data-integrator.html.
