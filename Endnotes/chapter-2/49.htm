StarfishETL; 2023; is an integration Platform as a Service (iPaaS), and more.  Cloud hosted services make StarfishETL adaptable beyond integration for migration, data governance, and data cleansing use cases. StarfishETL provides low-code mapping and scripting manage, personalize, and manipulate data at scale. Capabilities include:

1.    Connectors (to native application API for >375 connectors);

2.    Maps (which are blueprints that outline data movement that reduces up to 70% of the work);

3.    Scripting (to apply custom C#, Python, VBScript, or JavaScript scripting to map data); and

4.    Monitoring and notifications.
