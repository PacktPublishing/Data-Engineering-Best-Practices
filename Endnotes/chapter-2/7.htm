Ab Initio; 2023; is a powerful data processing platform that can process complex enterprise data using sophisticated business logic at scale. Applications are built graphically and can run within a batch, streaming, in-memory, or microservice architecture. Applications run in parallel and are distributed across containers or on different servers. Checkpoint restarts are supported as is application monitoring with alerting. The tool adapts to changes in data formats, rules, and infrastructure, providing business agility. The platform manages many foundational capabilities  for smooth IT operations. Refer to: https://www.abinitio.com/en.
