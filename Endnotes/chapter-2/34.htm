Microsoft SQL Server Integration Services (SSIS); 2023; is a platform for enterprise-level ETL. Example use cases are:

1.    copy or download files;

2.    load data warehouses;

3.    cleanse / mine data; and

4.    manage SQL Server objects and data.

It can ETL data from many sources such as XML data files, flat files, relational data sources, and then reverse ETL data into various destinations. You can use a GUI Integration Services tool to create solutions without code. Also, you can code ETL using the Integration Services Object Model to create packages, custom tasks and other package objects. Refer to: https://learn.microsoft.com/en-us/sql/integration-services/sql-server-integration-services?view=sql-server-ver16.
