Databricks Lakehouse Platform; 2023; is a modernized data warehouse. It combines data lakes and data warehouses to reduce costs and deliver on data and AI initiatives. It is built on open source and open standards. A lakehouse helps eliminate data silos which complicate data and AI processing. More details and an overview can be found here; What Is a Lakehouse? (https://www.databricks.com/blog/2020/01/30/what-is-a-data-lakehouse.html).
