AWS Glue; 2023; is a serverless data integration service.  It makes it easier to discover, prepare, move, and integrate data from various sources for analytics, ML, and development. It provides capabilities like:

1.    data integration;

2.    no infrastructure to set up/ manage;

3.    consumption based pricing;

4.    a AWS Glue Data Catalog for metadata storage;

5.    AWS Glue crawlers for data schema discovery via crawling;

6.    ETL development endpoints;

7.    ETL data preparation (matching and cleansing);

8.    debug development endpoints for editing, debugging, and testing; and

9.    Studio Job Notebooks for setting up serverless interactive sessions to save / schedule notebook code as AWS Glue jobs.

Refer to: https://aws.amazon.com/glue.
