Qlik Compose; 2023; capabilities include solutions to error prone ETL development efforts that are consumed with up to 60-80% of the time for infrastructure prep. A data model could become out-of-date before a BI project starts. Modifying brittle data warehouses cause delays, ties up resources and delays a projects ROI. Speeding TTM for analytics, the data warehouse creation and management lifecycle is streamlined. Qlik Compose for Data Warehouses is a modern approach that automates / optimizes data warehouse creation / operation by:

1.    designing the warehouse;

2.    generating ETL code; and

3.    quickly applying updates. BI project time, cost and risk is reduced.

Key Capabilities are:

1.    workflow designer and scheduler (to run data warehouse and data mart ETL tasks as a single, end-to-end process);

2.    lineage and impact analysis (to automatically create metadata during and then generate lineage as changes are implemented);

3.    monitoring and notification (to monitor status of all tasks / workflows);

4.    data profiling (to validate data when loading and identifying / repairing issues and discrepancies; and

5.    data quality (to configure and enforce pre-loading rules etc).
