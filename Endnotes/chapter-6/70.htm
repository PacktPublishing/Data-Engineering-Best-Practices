Elastic Observability; 2023; that is a solution that is built on the 'Elasticsearch, Logstash & Kibana (ELK)' stack to merge metrics, logs, traces, and profiling data to provide unified visibility.

Actionable AI-powered insights are provided.

It features an open, extensible, full-stack observability features built with AI and search technology to monitor (gain insights) across cloud-native and distributed systems.

Application, infrastructure, and user telemetry data for end-to-end observability is brought together in a single platform.

Detailed features are:

1.       'application performance monitoring (APM);'

2.       'infrastructure monitoring;'

3.       'real user monitoring;'

4.       'synthetic monitoring,' and

5.       'universal profiling.'

Refer to: https://www.elastic.co/observability
