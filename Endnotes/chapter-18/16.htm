"The Knowledge Graph Conference - Web-Scale Data Integration in Life Sciences & Healthcare Through Knowledge Graphs - Optum Health"; by Maulik R. Kamdar, Ph.D., Optum Health; August 30, 2023;

In this presentation Maulik states:

 

"And when we think about machine learning there are definitely some challenges in knowledge extraction as well. Because medical language in unstructured text itself is very complex there is a lot of specialization. There is a lot of negation.  There is context associated with certain relations … certain snippets (of text). There are  multiple entities. And so there is this work that we did in 2019 with a GloVe Embedding Model so it's before the advent of the Transformer Based Models and definitely before GPT. So bear in mind, some of these challenges are still evident." - Source: Maulik Kamdar, 2023.
