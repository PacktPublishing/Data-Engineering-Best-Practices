Voicebox core capabilities include:

1.      Knowledge Graph Question Answering democratizes enterprise analytics and data access. Voicebox answers natural language questions for data in a Stardog Knowledge Graph or Knowledge Catalog without writing graph queries BI tools;

2.      Knowledge Graph Data Modeling reduces cost for implementing knowledge graphs. Voicebox creates new knowledge graph ontologies from natural language inputs, CSV or other delimited files, or from any other Stardog Datasource. It can be used with Stardog Designer data model (ontology) lifecycle;

3.      Stardog Smart Help reduces operating costs. Voicebox answers natural language support questions about using, programming, or administering the Stardog EKG Platform.
