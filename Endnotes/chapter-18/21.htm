"Quetzal RDF"; by Julian Dolby (http://researcher.watson.ibm.com/researcher/view.php?person=us-dolby); IBM Thomas J. Watson Research Center - GitHub; Sep 17, 2019;

Quetzal is a SPARQL to SQL translation engine for multiple backends, such as DB2, PostgreSQL and Apache Spark. Quetzal was modified by Knoldus (now NashTech, see https://www.linkedin.com/company/knoldus) in order to work with Apache Ignite (Gridgain) as a backend in order to obtain blazing in memory graph performance for the ELSSIE knowledge graph.

Refer to https://github.com/Quetzal-RDF/quetzal.
