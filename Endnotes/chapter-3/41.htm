"What are practical projects for implementing zero trust?" by zScaler; 2023. IT security leaders will want to consider zero trust being part of an enterprise cloud-first strategy.

Gartner issued a report: "What Are Practical Projects for Implementing Zero Trust?" - by John Watts and Neil MacDonald; 17 March 2021 (https://www.gartner.com/en/documents/3999439) that has some critical answers.

Best practices for building zero trust include: how to create a secure common federated identity management system;  apply adaptive access for more granular resource access control; and roll out user-to-application segmentation ("Zero Trust Network Access - ZTNA." https://start.paloaltonetworks.com/ztna-for-dummies) .
