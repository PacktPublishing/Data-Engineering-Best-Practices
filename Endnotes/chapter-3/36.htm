Hiera - by Puppet; 2023. Puppet’s strength is reusable IaC. Code that serves many needs must be configurable with site-specific information in external configuration data files, rather than in the code itself.

Populating those files from a single source of truth security is provided by Hiera.

The tool may be extended to look up values in data source like: a CMDB, a PostgreSQL database, a custom web application, or another securely structured data file.

Refer to https://www.puppet.com/docs/puppet/8/hiera.html.
