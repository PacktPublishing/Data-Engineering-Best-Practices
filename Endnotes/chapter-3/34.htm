Terraform - by Terraform; 2023. HashiCorp’s Terraform is an IaC tool to define cloud and on-prem resources in human-readable configuration files that can be versioned, reuse, and shared.  Consistent workflows are used to provision and manage all infrastructure throughout its lifecycle.

Terraform manages low-level components like: compute, storage, and networking resources; and high-level components like DNS entries and SaaS features.

Refer to https://developer.hashicorp.com/terraform/intro.
