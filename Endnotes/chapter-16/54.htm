"Create a vector index in an Azure Machine Learning prompt flow (preview)"; by Microsoft;

The Azure Machine Learning services can be used to create a vector index from local files, folders, cloud storage, Azure Machine Learning data asset, Git repository, or SQL database. Various file types are supported like: .txt, .md, .pdf, .xls, and .docx. Lastly, you can use an existing Azure AI Search Index instead of creating new indices for ML purposes.

Refer to https://learn.microsoft.com/en-us/azure/machine-learning/how-to-create-vector-index?view=azureml-api-2.
