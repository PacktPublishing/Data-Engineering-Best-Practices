"Evaluating Naive RAG and Advanced RAG pipeline using langchain v.0.1.0 and RAGAS"; by Plaban Nayak in AI Planet; February 11, 2024.

In this article Plaban, the author, defines RAG and a way to evaluate solutions built with RAG LLMs. Ragas is used as the evaluation library which collects inputs, outputs, and context triplets to obtain metrics. The technology stack used in this article is as follows:

1.      LLM: OpenAI gpt-3.5-turbo as the LLM;

2.      Langchain: used to build the LLM application; and

3.      RAGAS: as the evaluation framework.

Refer to https://medium.aiplanet.com/evaluating-naive-rag-and-advanced-rag-pipeline-using-langchain-v-0-1-0-and-ragas-17d24e74e5cf.
