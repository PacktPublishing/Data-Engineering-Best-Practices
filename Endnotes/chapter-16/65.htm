LlamaIndex; 2024;

LlamaIndex is a framework for LLM applications which benefits from context augmentation. These are retrieval augmented generation (RAG) systems. LlamaIndex provides abstractions to ingest, structure, and access private or domain data to inject this data reliably into an LLM to drive accurate text generation output.

Refer to https://docs.llamaindex.ai/en/latest.
