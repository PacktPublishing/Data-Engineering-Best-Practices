Grid Search; by Wikipedia; 2024;

Grid Search is a conventional type hyperparameter optimization. It is also called parameter sweep, or the exhaustive searching through a manually specified subset of the hyperparameter space of an ML algorithm. A grid search must be guided by performance metrics, typically measured by cross-validation on the models training set or on a hold-out validation set.
