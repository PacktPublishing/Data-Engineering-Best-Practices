Genrocket; by Genrocket; 2024;

Genrocket provides synthetic test data automation capabilities.

The platform offers ways to automate the setup of the data model as a test data project and support multiple versions of the project.  With life cycle support the platform offers automations to detect and update changes to the data model.

With Genrocket's  self-service platform, synthetic data is designed by software test engineers and developers based on their individual test case criteria. Synthetic data is generated on-demand for each automated test run (see "Synthetic Data Generation" https://www.genrocket.com/synthetic-data-generation).

Refer also to https://www.genrocket.com/enterprise-features.
