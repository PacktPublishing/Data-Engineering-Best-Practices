Tumult Analytics; by Tumult Analytics; 2024;

Tumult Analytics is a Python library for computing aggregate queries on tabular data using differential privacy. A tutorial in Google Colab is available here: "Tumult Analytics walkthrough" (https://colab.research.google.com/drive/18J_UrHAKJf52RMRxi4OOpk59dV9tvKxO) Although Tumult Analytics is not strictly a synthetic data generator, it was included in order to round out the list of privacy preserving approaches when working with analytics.
