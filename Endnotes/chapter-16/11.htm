MLOps is defined as:

 

"MLOps stands for Machine Learning Operations. MLOps is a core function of Machine Learning engineering, focused on streamlining the process of taking machine learning models to production, and then maintaining and monitoring them. MLOps is a collaborative function, often comprising data scientists, DevOps Engineers, and IT." - Source Databricks, 2024

 

Refer to https://www.databricks.com/glossary/mlops.
