Googles What-If Tool; by Google; 2024;

The What-If Tool is an open source tool: Github What if Tool (https://pair-code.github.io/what-if-tool) used to understand ML model performance.  It was produced by the Google Tensorflow team as an interactive UI to visualize datasets and better understand outputs of TensorFlow models. It has grown to support more.

The What-If Tool supports:

1.      Binary classification;

2.      Multi-class classification; and

3.      Regression tasks.

Note that fairness optimization is available only with binary classification models.

Refer to https://cloud.google.com/blog/products/ai-machine-learning/introducing-the-what-if-tool-for-cloud-ai-platform-models.
