"How to Evaluate the Performance of PyTorch Models"; by Adrian Tam; in Deep Learning with PyTorch; April 8, 2023;

In this article the author correctly points out that it is important to use scientific methods to evaluate the performance of your neural network / deep learning models. The same method is necessary when comparing other kinds of machine learning models. The article provides an example workflow to robustly evaluate model performance. The examples use PyTorch in order to build models. The objective of the post is to teach you how to:

1.      Evaluate a PyTorch model using a verification dataset;

2.      Evaluate a PyTorch model with k-fold cross-validation.

Note: Adrian Tam authored: "Deep Learning with PyTorch" (Deep Learning with PyTorch).
