IBM Well-Architected Framework; 2023;

IBM's Well-Architected Framework is a structured collection of materials to help architects create hybrid, secure, resilient, and performant cloud solutions. There are six pillars:

1.      hybrid and portable;

2.      resiliency;

3.      efficient operations;

4.      security and compliance;

5.      performance; and

6.      financial operations and sustainability.
