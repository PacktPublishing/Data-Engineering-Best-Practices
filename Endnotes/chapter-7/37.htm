Understanding TestOps Best Practises and Working Architecture; December 9, 2022; by Navdeep Singh Gill;

It is the ability to perform QA using DevOps actions while leveraging  CI/CD for continuous testing. Performing:

1.      functional;

2.      non-functional;

3.      performance; and

4.      all other testing in an automated manner is the goal.

Infrastructure, security and compliance, database administration, network configuration and other tasks are part of the test cycle.

TestOps is a framework that continuously tests and validates the software at all levels. The framework integrates CI/CD automated tests in and the products quality lifecycle. And the framework can be combined with tools like: Selenium,  Docker,  BrowserStack, and Jenkins to make tests robust and performant.
