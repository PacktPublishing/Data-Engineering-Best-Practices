Mirantis Kubernetes Engine; 2023;

Mirantis Kubernetes Engine is formerly Docker Enterprise, and was acquired by Mirantis in November 2019;

Mirantis is a simple, flexible, and scalable container orchestration and enterprise container management.

It provides you to build, run, and scale cloud native applications.

It increases developer efficiency and release frequency while reducing costs.

And it enables you to deploy Kubernetes or Swarm clusters out of the box and manage them via API, CLI, or web interface.

Lasty, it supports simplified cluster management with integrated role-based access control (RBAC) and identity management.
