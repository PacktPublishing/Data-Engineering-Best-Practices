Enterprise Business Intelligence (Analytics) on Azure; 2024;

In Enterprise Business Intelligence (Analytics) on Azure data is sourced from an Azure SQL Server database.

Data ingestion and storage take place in the Azure Data Lake Gen2 used as a staging area during ingestion.

PolyBase may be used to copy data into an Azure Synapse dedicated SQL pool.

The distributed  Azure Synapse Analytics service is designed to perform large data analytics since it implements a massive parallel processing (MPP), design needed for running high-performance analytics.

The Azure Synapse dedicated SQL pool is the target for subsequent ingestion activities from a source like an on-premises system.

The SQL pool can be used for internal data factory processing or for serving data directly through Power BI DirectQuery. Azure Pipelines is used to orchestrate various data ingestion and transformation processing steps within your Azure Synapse workspace.

Refer to: Azure Enterprise Business Intelligence Architecture.
