LucidScale; 2023;  LucidScale is a cloud visualization solution that helps organizations see and understand cloud environments.

AWS, Azure, and GCP are supported.

You are able to see key cloud governance data in context and understand elements that may be orphaned or unidentified.

This helps you avoid risk and cut unnecessary costs.

Filters are used to enable you to focus on specific elements such as: security groups, VPCs, Virtual Machines, Zones, etc.

You are able to generate or update cloud diagrams to confirm what was built matches your intent.

This avoids downtime and prevents holes in security and / or infrastructure.

You also may validate essential infrastructure details such as: instance sizes, regions, data types / locations, and with that any security concerns which are now identifiable.
