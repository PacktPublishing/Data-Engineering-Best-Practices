Tanzu; 2023;

Tanzu by VMWare is a modular, cloud native application platform that accelerates development, delivery, and operations across multiple clouds.

It is a:

1.      Single, integrated cloud native application platform with templates, best practices, and integrated tooling providing flexibility for in place applications;

2.      Consistent developer experience enabling developers to be productive, deliver value with streamlined, self-service developer experience for Kubernetes with automated infrastructure packaging, and security;

3.      End-to-end secure platform for DevOps /SecOps approaches with controls to secure contents of container images, paths to production, and container production interactions across clusters; and

4.      Cloud neutral tool supporting cloud infrastructure with templates, automation, and role-based Uis; and

5.      Centralized, automated, and secure operating model for modern apps and Kubernetes on private and public clouds.
