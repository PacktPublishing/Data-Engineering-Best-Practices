Holori; 2023;

Holori allows you to design cloud diagrams, visualize cloud infrastructures, estimate costs, and then deploy the solution.

GCP, AWS, and Azure are supported.  You can create / import cloud diagrams then visually design your diagram using cloud vendors official and latest icons, components, and connectors.

You do not have to start from scratch, but rather choose a template from the library.

After connecting your cloud account, you can easily import your existing infrastructure making it humanly readable.

You can also import a terraform script, to create your cloud diagram from prior developed IaC scripting.

An estimate of the cost of each item in your cloud is available in order to give accurate estimations of overall cost. As a diagram or as cloud deployment; output options exist that make the tool attractive.
