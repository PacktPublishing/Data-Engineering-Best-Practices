Mindgrammer; 2023;

Mindgrammer produces a product called: Diagrams that lets you draw the cloud system architecture in Python code.

It began as a prototyping tool for new system architectures that lacked design tools. You can use it to describe the existing system architecture.

Diagram as Code allows you to track architecture diagram changes in any version control system.

Diagrams supports: AWS, Azure, GCP, Kubernetes, Alibaba Cloud, Oracle Cloud and on-premises nodes, SaaS and major frameworks and languages.

It does not control any actual cloud resources, nor does it generate cloud formation or terraform code.

It is used for drawing cloud system architecture diagrams.

Its free open source and since it is python scripted it can be generated from cloud provider API output, but it requires a high degree of coding to develop and then integrate.
