Best practices framework for Oracle Cloud Infrastructure; 2023;

Best practices framework for Oracle Cloud Infrastructure are best practices to providing infrastructure and platform services for enterprise workloads.

In each service provides features with recommendations and best practices to design and operate cloud topologies that deliver maximum business value.

There are four business goals:

1.      security and compliance;

2.      reliability and resilience;

3.      performance and cost optimization; and

4.      operational efficiency.
