Microsoft SPEC#, 2023;

Spec# is a formal C# extension language for API contracts for non-null types, preconditions, postconditions, and invariants. It is a programming methodology.

It works even when code is built with callbacks and multi-threading.

It is also a Microsoft research tool that explores formal specifications for dynamic / static tools.

Spec# is on Github here: (https://github.com/microsoft/CodeContracts/blob/master/Microsoft.VisualStudio.CodeTools/Setup/SpecSharp.makefile).

Refer to https://www.microsoft.com/en-us/research/project/spec.
