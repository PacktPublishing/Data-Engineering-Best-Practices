Java Modeling Language (JML), 2023;

JML is a behavioral interface specification language for Java modules.

JML formally describes the behavior of a Java module that implements DbC goals with Java programming. Specifications may be written in Java program files as annotations or placed into specification files. Comment placed annotations (aka JML annotations) if in a Java file are preceded by an @. For example: //@ <JML specification>

Refer to https://en.wikipedia.org/wiki/Java_Modeling_Language.
