Jenkins on Google Cloud;

Jenkins may be deployed on GCP as depicted in the following diagram.

Note:

1.      A GCP service account needs to be added to Jenkins for it to be able to create agent instances and push artifacts to GCP Cloud Storage;

2.      Jenkins provisions instances on demand while running CI/CD builds; and

3.      Old artifacts move through GCP storage classes in order to limit any retention costs:

Refer to https://www.jenkins.io/doc/tutorials/tutorials-for-installing-jenkins-on-Google-Cloud and https://cloud.google.com/architecture/using-jenkins-for-distributed-builds-on-compute-engine?hl=en.
