OpenTelemtry; 2023;

OpenTelemtry (or OTel for short), is an open source observability framework for:

1.      Instrumenting ;

2.      Generating;

3.      Collecting; and

4.      Exporting IT operationally gathered telemetry data.

It is an industry-standard supported by >40 observability vendors.

It can be used with observability backends like: Jaeger (https://www.jaegertracing.io) and Prometheus.

Refer to https://opentelemetry.io.
