Amazons CloudWatch; 2023;

CloudWatch is Amazon's APM tool.

The display below contains an overview. Items refreshed automatically. In the:

1.      upper left: is a list of AWS services in the account and alarms associated with those services;

2.      upper right: shows either 2 or 4 account alarms based on services in use with their ALARM state;

3.      upper areas enable assess to AWS service health to help diagnose issues;

4.      default dashboard;

5.      custom dashboards to show metrics that you created for custom services / applications, or additional key metrics from various cloud services to be monitored; and

6.      automatic cross-service dashboard for all AWS services used (used for cloud service drill down).

Refer to https://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/GettingStarted.html.
