FluentBit; 2023;

FluentBit is a fast, lightweight, scalable logging and metrics processor / forwarder.

It is a telemetry agent for observability data from: Linux, macOS, Windows, and BSD OS.

It is used in cloud and containerized environments.

In FluentBit V2, there are new integrations for:

1.       Open Telemetry (OTel);

2.      Wasm (https://docs.fluentbit.io/manual/development/wasm-input-plugins); and

3.      Logs to Metrics (https://docs.fluentbit.io/manual/pipeline/filters/log_to_metrics); and

4.      others.

Refer to https://fluentbit.io.
