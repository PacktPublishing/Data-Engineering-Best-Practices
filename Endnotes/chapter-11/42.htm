Loupe; 2023;

Loupe is able to understand how an application is deployed. Production from development environment data can be separated as well as logs for each component within each environment.

Diagnostic data is gathered for .NET or Java applications and analyzed to identify errors with notifications being sent via email, Slack, or other method.

The tool understands agile development workflows and when fixes have been deployed for previously identified errors.
