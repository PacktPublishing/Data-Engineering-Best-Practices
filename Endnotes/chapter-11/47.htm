Scouter; 2023;

Additional Scouter documentation is available here: https://github.com/scouter-project/scouter/blob/master/scouter.document/index.md.

What makes Scouter special is its unique point of view and powerful built in features.

The tool must be installed inside your applications security zone so that collection of a rich set of data of target system data is possible.

SaaS services do not have this access level. Scouter uses the Eclipse RCP platform to aggregate data together.

Scouter can analyze stacks together to solve performance problems via the Stack Frequency Analyzer feature.

Refer to https://github.com/scouter-project/scouter.
