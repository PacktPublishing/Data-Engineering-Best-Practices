Travis CI; 2023;

Travis CI is a powerful tool for continuous integration.

Connections exist to: Assembla, and GitHub (as well as others). Travis CI has a Build Matrix for parallel building. Configuration is possible via a simple syntax to support faster time to value (TTV).

Ability to build from clean VMs. Live build requests are supported as are auto-deployments for passing builds.

Pull requests are also supported.  Test can be performed on:

1.      Mac;

2.      Linux; and

3.      iOS with additional tests performed on unique architectures like:

a.    AMD Graviton 64;

b.    IBM PowerPC; and

c.     IBM Z.

Refer to https://www.travis-ci.com.
