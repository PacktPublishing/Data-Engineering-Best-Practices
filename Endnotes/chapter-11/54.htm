Splunk; 2023;

Splunk offers a solution to collect all trace data instead of sampled subsets.

Splunk APM correlates cloud infrastructure and the microservices log data.

An AI-driven approach sifts through trace data to highlight microservices responsible for errors.

Splunk provides you with the capability to segment transaction data based on many factors including the cardinality of running resources.

Opentelemetry integration is available.

Refer to https://www.splunk.com/en_us/products/apm-application-performance-monitoring.html.
